# Memory-Game 
A memory game project for full-stack course
focusing on the fetch method and working with API

## Technology and the target of this project
this was made for me to practice working with api
this was made in vscode using html, javascript and css
all the images of the cards are taken from three different free API's
that have been provided to me by the teacher

## opening the game
in order to check my project you should download the whole folder
and you can either click on the "index.html" file or open it in vscode
and open it with live server

## Prerequisites
make sure you have vscode and the live server extention
